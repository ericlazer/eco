Quick Overview

Snowball Money provides a universal naming service that works across all major blockchains, replacing complex addresses with human-readable names like 'alice.snow' for easy crypto transfers.

Unlike traditional single-chain name services, Snowball's modular architecture allows one name to resolve to addresses on multiple chains, simplifying the multi-chain user experience.

Key Features

  • Multi-chain name resolution
  • One name for all addresses
  • Subdomain management
  • Integration with major wallets
  • Decentralized profile pages
  • Name marketplace

How to Use Snowball Money

Prerequisites

  • Wallet on supported chains
  • Small amount for registration fee
  • Unique name availability

Getting Started

  1. 1

    Search Names

    Visit snowball.money and search for your desired .snow name availability.

  2. 2

    Register Name

    Pay registration fee (usually $5-20/year) to claim your name.

  3. 3

    Set Addresses

    Configure which addresses your name resolves to on each blockchain.

  4. 4

    Create Profile

    Optionally create a decentralized profile page at yourname.snow.

  5. 5

    Use Everywhere

    Share your .snow name instead of addresses for all crypto transactions.

Tips & Best Practices

  • Register common names early
  • Set up all chain addresses at once
  • Enable IPFS hosting for profiles
  • Consider multi-year registration discounts

Frequently Asked Questions

Can I transfer my name?

What if I forget to renew?

30-day grace period after expiration before names become available to others.
